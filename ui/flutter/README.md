# Sahool Flutter Application

تطبيق Flutter كامل لنظام سهول للمراقبة الزراعية.

## المميزات

- ✅ إدارة المزارع والحقول
- ✅ عرض صور الأقمار الصناعية
- ✅ تحليل NDVI وعرض الخرائط
- ✅ رسم الحقول على الخريطة
- ✅ عرض التنبيهات والتقارير
- ✅ رسوم بيانية للتغيرات الزمنية
- ✅ مقارنة الصور قبل وبعد
- ✅ توليد تقارير PDF

## المتطلبات

- Flutter SDK 3.0.0 أو أحدث
- Dart SDK 3.0.0 أو أحدث

## التثبيت

1. تثبيت الحزم:
```bash
flutter pub get
```

2. تشغيل التطبيق:
```bash
flutter run
```

## الإعدادات

قم بتعديل عنوان API في ملف `lib/main.dart`:

```dart
Provider<ApiService>(
  create: (_) => ApiService(baseUrl: 'http://YOUR_API_URL:8000'),
),
```

## الاستخدام

### إضافة مزرعة جديدة
1. افتح صفحة "المزارع"
2. اضغط على زر "إضافة مزرعة"
3. أدخل البيانات المطلوبة

### رسم حقل جديد
1. افتح صفحة "رسم الحقول"
2. اختر شكل الحقل (دائرة، مستطيل، مضلع)
3. ارسم الحقل على الخريطة
4. احفظ الحقل

### عرض تحليل NDVI
1. افتح صفحة "الحقول"
2. اختر حقلاً
3. اضغط على "جلب صور الأقمار"
4. بعد اكتمال التحميل، اضغط على "معالجة NDVI"
5. شاهد النتائج والخرائط
